#SQL
---เรียกดูข้อมูลในตาราง---
A. เรียกดูข้อมูลทั้งหมดใน customers =>
   select * from customers ;
B. เรียกดูข้อมูลที่ไม่ซ้ำกัน =>
   select distinct * from customers ;
C. เรียกเฉพาะคอลัมน์ Country =>
   select Country from customers ;
D. เรียกดูข้อมูลเฉพาะ"คอลัมน์ที่ข้อมูลไม่ซ้ำกัน" =>
   select distinct Address from customers;
E. ตารางในฐานข้อมูลเรียกว่า =>  
   ชุดข้อมูลที่มีโครงสร้าง จัดเรียงเป็นแถวและคอลัมน์
F. คีย์เวิร์ดใน SQL คำนึงถึงตัวพิมพ์ใหญ่และตัวพิมพ์เล็กหรือไม่? =>
   No, they are not case-sensitive.
G. เหตุใดจึงต้องใช้เครื่องหมายเซมิโคลอน (;) ไว้ท้ายคำสั่ง SQL? => 
   เพื่อแยกคำสั่ง SQL หลายคำสั่งออกจากกัน
